Help for comparison of TAD partitions

Input

Input files should be text files in BED-like format (chr-start-end). The tool can be used with provided demo files.

Set the parameters for similarity calculation in the left panel.

Output

Overview of the loaded data in the "Loaded data" tab.

Descriptive features of the input data in the "Dataset overview" tab.

Results of the similarity metric calculation in the "Similarity table" (data frame) and "Similarity heatmap" (heatmap) tabs. They can be downloaded in "txt" and "png" format respectively.

Similary metric calculation

Following metrics are currently implemented:

  • Measure of Concordance (MoC) (warning: can be slow)
  • Jaccard index (bins)
  • Jaccard index (boundaries)
  • Ratio of matching TADs
  • Variation of information

See the tab "Details about the metrics" for more information.
